# Cyber Terminal Toolkit

A polished Astro landing page for the Cyber Terminal project. This repository packages the website experience for the toolkit with cinematic terminal-inspired visuals, interactive 3D sections, and a GitHub-ready project structure.

## Overview

Cyber Terminal Toolkit is a static marketing site built to showcase a GPU-accelerated, cross-platform terminal concept. The site highlights the product story, architecture, built-in tools, supported platforms, and open-source positioning in a presentation that is fast to deploy and easy to maintain.

## Highlights

- Astro-powered static site with component-based structure
- Three.js and GSAP driven interactive visuals
- Terminal-inspired UI with responsive sections and motion
- Clear product storytelling for features, architecture, tools, and platforms
- Ready for GitHub hosting and further customization

## Tech Stack

- Astro
- TypeScript
- Three.js
- GSAP
- CSS

## Getting Started

### Prerequisites

- Node.js 18 or newer
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open the local dev server URL shown in the terminal, usually `http://localhost:4321`.

### Production Build

```bash
npm run build
```

### Preview Production Output

```bash
npm run preview
```

## Scripts

- `npm run dev` starts the Astro development server
- `npm run build` creates the production build
- `npm run preview` previews the built site locally
- `npm run astro` runs Astro CLI commands directly
